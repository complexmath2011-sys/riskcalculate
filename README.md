# Risk Calculator

Простой калькулятор риска для трейдеров.

Позволяет рассчитать размер сделки на основе:
- суммы, которую пользователь готов потерять
- процента риска

## Как это работает

1. Пользователь вводит сумму риска (USDT)
2. Указывает процент риска
3. Получает размер сделки, при котором указанный процент равен сумме риска

Формула:
Размер сделки = Сумма риска / (Риск% / 100)


## Где используется

Калькулятор предназначен для использования в Telegram в формате Web App.
Оптимизирован для мобильных устройств.
